# relq_ecommerce

## Description

This responsive webbsite allow you to choose an item, read the description, add or remove it from the cart, as well as choose the quantity and calculate the price

## Frontend Tecnologies

* React
* Redux
* Axios
* CSS 3

## Backend Tecnologies

* Node.js
* Express.js
* MongoDB



## Features

* Home screen
```
    Logo
    Cart(button)
    Shop(button)
    View(button)
    Link
    Hamburger
```
* Cart screen
```
    Procead To Checkout
    Items qty(button)
    Delete(button)
```
* Product screen
```
    Description
    Price
    Add to cart(button)
    Qty(button)
```
* backend
```
    Crud
    Seedeng
```
