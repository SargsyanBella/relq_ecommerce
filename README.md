# relq_ecommerce

## Description

This responsive webbsite allow you to choose an item, read the description, add or remove it from the cart, as well as choose the quantity and calculate the price

## Tecnologies

* react
* redux
* axios
* CSS 3
## Features

* Home screen
```
    Logo
    Cart(button)
    Shop(button)
    View(button)
    Link
    Hamburger
```
* Cart screen
```
    Procead To Checkout
    Items qty(button)
    Delate(button)
```
* Product screen
```
    Description
    Price
    Add to cart(button)
    Qty(button)
```

### Installing
```
git clone https://github.com/SargsyanBella/relq_ecommerce.git
cd  project MERN_ECOMMERCE
npm install
```

## Getting Started
